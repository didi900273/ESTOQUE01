💡 Como funciona — Explicação simples
Por que os dados ficam salvos?
O sistema guarda tudo no localStorage do seu browser — é uma memória interna do Chrome/Edge que nunca apaga sozinha, nem quando fecha o site. Os dados estão seguros enquanto não limpar o histórico do browser.

Como partilhar com o chefe?
1. Você clica em Exportar .json → baixa um ficheiro
2. Envia o ficheiro por WhatsApp ou e-mail ao chefe
3. O chefe abre o site → vai em Compartilhar → clica Importar .json
4. Pronto — ele fica com os mesmos dados que você!

Quando repetir?
Sempre que fizer alterações importantes (novos itens, saídas, etc.) e quiser que o chefe veja os dados actualizados.
