Para que o sistema seja partilhado entre você e o seu chefe, os dados ficam guardados na nuvem.

1

2
No painel, vá em API Keys → Create Access Key e copie a chave que começa com $2b$...
3
Cole a chave abaixo e clique em Criar e Ligar. O sistema cria automaticamente a base de dados e você recebe um ID para partilhar.
